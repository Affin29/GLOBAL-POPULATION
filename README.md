# GLOBAL-POPULATION
*Overview
The Global Population Dataset provides country-level demographic information including population size, age distribution, gender split, and population density.
It is ideal for EDA, Power BI dashboards, and research on global demographic trends.

*Dataset Description
Columns Included
Country – Name of the country
Population (in millions)
Female Population (in millions)
Male Population (in millions)
Sex Ratio (males per 100 females)
Population Aged 0–14 (%)
Population Aged 60+ (%)
Population Density
Youth_to_Senior_Ratio

*Use Cases
This dataset is suitable for:
Exploratory Data Analysis
Statistical and demographic studies
Power BI dashboards
